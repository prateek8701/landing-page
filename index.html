Understanding Media Queries

Media queries are the foundation of responsive web design. They allow us to apply different CSS styles based on device characteristics like screen width, height, and orientation.

üì± Mobile Small 320px - 480px
The smallest screens including older smartphones and devices in portrait mode. Focus on single-column layouts, larger touch targets, and simplified navigation.
/* Mobile Small Devices */ @media (max-width: 480px) { .hero h1 { font-size: 2rem; } .nav-links { flex-direction: column; } section { padding: 2rem 1rem; } }
üì± Mobile Large 481px - 767px
Modern smartphones in portrait and landscape modes. Slightly larger screens allow for more content while maintaining mobile-friendly design patterns.
/* Mobile Large Devices */ @media (min-width: 481px) and (max-width: 767px) { .hero h1 { font-size: 2.5rem; } .demo-grid { grid-template-columns: 1fr; } .mobile-menu { display: flex; } }
üì± Tablet 768px - 991px
Tablet devices and small laptops. These screens can accommodate two-column layouts and more complex navigation structures while remaining touch-friendly.
/* Tablet Devices */ @media (min-width: 768px) and (max-width: 991px) { .demo-grid { grid-template-columns: repeat(2, 1fr); } .features-grid { grid-template-columns: 1fr; } .hero h1 { font-size: 3rem; } }
üíª Desktop 992px - 1199px
Standard desktop and laptop screens. Multi-column layouts work well here, and you can utilize horizontal space effectively with sidebars and complex grids.
/* Desktop Devices */ @media (min-width: 992px) and (max-width: 1199px) { .demo-grid { grid-template-columns: repeat(3, 1fr); } nav { padding: 1rem 1.5rem; } .nav-links { display: flex; } }
üñ•Ô∏è Large Desktop 1200px+
Large monitors and high-resolution displays. Take advantage of the extra space with wider layouts, larger typography, and more detailed content arrangements.
/* Large Desktop Devices */ @media (min-width: 1200px) { .hero h1 { font-size: 4.5rem; } .demo-grid { grid-template-columns: repeat(4, 1fr); } .container { max-width: 1400px; } }
